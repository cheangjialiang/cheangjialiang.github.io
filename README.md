# cheangjialiang.github.io

Personal website for Leo Cheang.

- Live: https://cheangjialiang.github.io/
- CV: `public/Leo's CV.pdf`

## Local development

Prereqs: Node.js 20.19+ and Bun 1.2+

```bash
bun install
bun run dev
```

## Deployment

Pushing to `master` (or `main`) auto-deploys via GitHub Actions to GitHub Pages.
